# ✦ مولّد البروميت الاحترافي (Professional Prompt Builder)

تطبيق ويب تفاعلي يحوّل **فكرتك الأولية** إلى **بروميت احترافي جاهز للنسخ** لاستخدامه في Google AI Studio أو Codex أو أدوات Vibe Coding، عبر ٥ أسئلة موجّهة متعددة الاختيارات — ويولّد بروميتاً يتكيّف مع نوع المخرج (منصة / تقرير / صورة / مبادرة).

## 🎯 المزايا

- **مولّد ذكي بخطوات:** إدخال الفكرة ← ٥ أسئلة اختيارات متعددة ← بروميت نهائي + مقترح احترافي منظّم (٢٠ نقطة).
- **يتكيّف مع نوع المخرج:** منصة/تطبيق، تقرير، صورة/إنفوجرافيك، مبادرة — أو مزيج موحّد.
- **مكتبة برومبتات:** حفظ، بحث، عرض، نسخ، حذف.
- **لوحة إحصائيات:** عدد البرومبتات والتوزيع حسب نوع المخرج (رسم بياني).
- **تصدير:** نسخ فوري، `.txt`، `.md`، وطباعة/حفظ PDF.
- **افتراضات مهنية:** عند نقص الإجابات يكمل النموذج بأفضل الممارسات ويصرّح بها.
- **واجهة عربية RTL**، وضع ليلي/نهاري، تصميم متجاوب (جوال/تابلت/شاشات كبيرة).

## 🏗️ البنية التقنية

| الطبقة | التقنية |
|--------|---------|
| الواجهة الأمامية | HTML + CSS + JavaScript (بدون أطر، بلا خطوة بناء) |
| الخلفية (اختيارية) | Node.js + Express — REST API |
| قاعدة البيانات | **SQLite** (`better-sqlite3`) محلياً + **LocalStorage** كطبقة عميل |

> التطبيق يعمل بوضعين تلقائياً:
> - **وضع محلي (افتراضي):** LocalStorage — يعمل مباشرة على GitHub Pages وبدون خادم.
> - **وضع الخادم:** عند تشغيل مجلد `backend` يكتشفه التطبيق تلقائياً ويحفظ في SQLite.

## 🚀 التشغيل

### 1) الواجهة فقط (الأسرع)
افتح `index.html` مباشرة في المتصفح، أو شغّل خادماً ثابتاً:

```bash
# من جذر المشروع
python3 -m http.server 8080
# ثم افتح http://localhost:8080
```

### 2) النسخة الكاملة (واجهة + خلفية + SQLite)

```bash
cd backend
npm install
npm start
# افتح http://localhost:4000
```

سيقدّم الخادم الواجهة الأمامية و REST API معاً، وستُحفظ البرومبتات في `backend/prompts.db`.

```

│   ├── server.js             # خادم Express
│   ├── db.js                 # طبقة SQLite
│   ├── package.json
│   └── prompts.db            # (يُنشأ تلقائياً)
└── README.md
```

## 🔌 REST API

| Method | Endpoint | الوصف |
|--------|----------|-------|
| GET | `/api/health` | فحص الصحة |
| GET | `/api/prompts` | كل البرومبتات |
| GET | `/api/prompts/:id` | بروميت واحد |
| POST | `/api/prompts` | إنشاء/تحديث |
| DELETE | `/api/prompts/:id` | حذف واحد |
| DELETE | `/api/prompts` | حذف الكل |
